##### 使用案例

?> 以Mysql同步到Hive为例

```sql
CREATE TEMPORARY VIEW users_view
USING org.apache.spark.sql.jdbc
OPTIONS (
    driver 'com.mysql.cj.jdbc.Driver',
    url 'jdbc:mysql://dcloud-dev:30102/ispong_db',
    user 'ispong',
    password 'ispong123'
    dbtable 'users',
);

insert into hive_db.users select * from users_view where age > 15
```
